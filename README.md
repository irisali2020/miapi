# API RESTfull

Este proyecto ha sido desarrollado con proposito de aprendizaje. Su objetivo es hacer una API REST desde la creación del servidor utilizando EXPRESS, con una arquitectura ordenada donde se identifican facilmete las carpetas: routes, controllers, models, middleware, utils.

Está conectado a una base de datos creada en FIREBASE / FIRESTORE.

El CRUD está funcionando y fue probado con los test para la aplicación y los productos.

Se agregó la protección de las rutas, cumpliendo con los requerimientos de entrega del proyecto final.
